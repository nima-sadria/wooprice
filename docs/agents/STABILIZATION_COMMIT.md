# Stabilization Commit

Before commit provide:

* git status --short
* git diff --name-only
* git diff --cached --stat

After commit provide:

* commit hash
* files committed
* final git status

Rules:

* Commit only intended files.
* Never use git add .
* Never use git add -A.
* Verify staged files before commit.

Do not deploy.

Do not start next phase automatically.
